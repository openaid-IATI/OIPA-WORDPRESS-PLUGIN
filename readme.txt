The plugin lets you configure WP parameters needed for the integration of the OIPA Search API

== Description ==

The plugin helps the WP admin to set up certain important parameters that are required for the
correct implementation of the OIPA Search API.

It is important to note that the plugin has to be installed and configured first, before the
OIPA WP theme is installed.
 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the folder `oipa` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the OIPA Search params through the 'Settings/OIPA' menu in WordPress
